## 📖 项目背景
本项目为 2025 年全国大学生电子设计竞赛 G 题《电路模型探究装置》的 FPGA 核心控制逻辑库。
系统旨在设计一款可完全脱离外部仪器的智能探究装置，能够产生可调节的探测信号加给已知/未知电路模型，并在 2 分钟内完成未知电路的自主学习与特征提取。

为突破单一 MCU 的实时调度与高频采样瓶颈，本项目创新性采用 **FPGA + ARM (STM32) 异构双核控制架构**。本仓库开源了该系统中运行于 Xilinx ZYNQ-7010 内的全部 FPGA 逻辑代码。

## ✨ 核心特性与数字处理引擎

本 FPGA 工程主要包含以下三大高实时性数字信号处理核心：

### 1. 多波形 DDS 高精度信号源 (`DDS_AD9767.v`)
* **频率合成：** 支持输出正弦波、方波、三角波及直流信号，频率范围覆盖 100Hz 至最高不小于 1MHz。
* **动态幅度校准：** 在 FPGA 内部实现基于 4V 参考电压基准的查表与动态移位乘法缩放，输出信号峰峰值最大可达 3V。

### 2. 32位高精度可重构 IIR 滤波器 (`iir_biquad_32bit.v`)
这是应对未知黑盒电路重构的核心模块，用于接收 ARM 计算的数字滤波器模型参数并在 FPGA 端硬件化落地：
* **Direct Form I 结构：** 基于标准的二阶差分方程硬件化映射。
* **高精度防发散设计：** 内部乘法器与数据通路全面升级为 **32位有符号数（Q4.28 定点数格式）**，彻底解决截断误差导致的极点漂移及系统发散问题。
* **饱和限幅防震荡：** 针对 14-bit DAC，设计严格的边界限幅保护（`0` ~ `16383`），保障最终输出波形绝对平滑。

### 3. 高鲁棒性数模接口与通信栈
* **ADC 高频采样 (`adc.v`)：** 提供驱动 AD9238 的 12-bit 数据稳定锁存与 4MHz 采样时钟。
* **跨时钟域串口通信 (`uart_128.v` / `top.v`)：** 部署 AXI4-Stream UART 协议栈，并使用影子寄存器缓存 21 字节的长配置帧（1 字节帧头 + 20 字节滤波器系数），实现 ARM 与 FPGA 之间极低延迟与无损交互。

## 🗂️ 目录结构说明

| 文件名 | 功能描述 | 核心技术要点 |
| :--- | :--- | :--- |
| `top.v` | 系统顶层模块 | 全链路时钟路由、UART 帧数据解析与状态机状态分发 |
| `iir_biquad_32bit.v` | IIR 滤波器运算核心 | Q4.28 格式定点数乘加树、饱和限幅逻辑 |
| `DDS_AD9767.v` | DDS 频率幅度映射层 | 极其详尽的频率控制字（Fword）和幅度比例控制字（Amp_scale）状态机 |
| `DDS_Module.v` | DDS 波形发生底层 | 相位累加器、基于 ROM 查表的波形生成及数据中心偏移处理 |
| `key_filter.v` | 硬件级按键消抖 | 基于 125MHz 时钟和 20ms 延时计时器的跨时钟域防抖状态机 |
| `sync_signal.v` | 跨时钟域同步 | 消除亚稳态的两级寄存器打拍同步 |
| `divider.v` | 分频器模块 | 支持动态分频系数与占空比配置的时钟分频 |

## 📊 硬件测试指标摘录 (摘自 B229 组设计报告)
* **输出精度：** 100Hz 步进频率正弦波输出误差 0%。
* **波形重构误差：** 生成推理模拟信号幅值与频率稳定，有源滤波电路与传递函数表征的输出电压幅度相对误差绝对值 ≤ 10%。
